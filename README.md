# Game Projects Portfolio

Репозиторий с учебными и личными игровыми проектами на C#/.NET.

## Projects

| Project | Description | Stack |
|---|---|---|
| [NPC Simulator](./npc-simulator) | Симулятор маленького города: NPC перемещаются по локациям, меняют настроение, хранят воспоминания и реагируют на действия игрока. | C#, WPF, .NET Framework 4.8 |
| [Echo Room](./echo-room) | Пазл-прототип, где действия игрока записываются и затем повторяются эхо-копиями прошлого. | C#, WPF, .NET Framework 4.8 |
| [NEON MEMERY](./neon-memery) | Неоновый интерактивный архив с таймлайном фрагментов памяти, ручной GDI+ отрисовкой и JSON-сохранениями. | C#, WinForms, GDI+, JSON |
| [Black Book](./black-book) | Карточная RPG: коллекция карт, колоды, магазин, миссии, награды и пошаговые бои с боссами. | C#, WPF, XAML, Newtonsoft.Json |
| [Tanks](./tanks) | Аркадный танковый экшен с собственной картой, ботами, снарядами, бонусами, монетами, базами и миссионной кампанией. | C#, WPF, WriteableBitmap |

## Notes

В репозитории оставлены исходники, файлы решений и проектные настройки. Сборочные папки `bin`, `obj`, локальные файлы IDE и восстановленные пакеты не добавлены.
